# Biketime Public (B2B E-shop)

Veřejný B2B e-shop na biketime.cz.

- **Stack:** Next.js 15, React 19, TypeScript, Tailwind 4, PostHog
- **Firebase:** Stejný projekt jako admin (`/Users/jiriadamcernik/Dev/biketime`)
- **Katalog:** Čte z kolekce `products_v2`
- **Uživatelé:** ShopUser s PriceLevel (A/B/C/D) a rolemi (poweradmin/admin/shop/pending)
